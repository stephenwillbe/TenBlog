# TenBlog

## 个人博客，后台+前台

##### 一个实战练习的项目，开源，可部署。

- **你可以把它作为个人网站，集成了个人首页，个人摄影作品，个人博客等功能；**
- **你也可以把它作为springboot技术的学习项目，My Site也足够符合要求，且代码和功能完备；**

### 后台技术
springboot框架

shiro安全框架

redis缓存

Mysql数据库

Maven项目管理

restful接口风格

### 前端技术
thymeleaf模板引擎

layui风格

echarts数据分析图

editormd富文本编辑插件
